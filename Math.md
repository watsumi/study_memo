# module Math
[ルリマ](https://docs.ruby-lang.org/ja/latest/class/Math.html)

>[要約]<br>
>浮動小数点演算をサポートするモジュールです。<br>
>Math モジュールにはさまざま数学関数がモジュール関数として定義されています。<br>
>モジュール関数は以下のように，モジュールの特異メソッドとして呼び出す使い方と、<br>モジュールをインクルードしてレシーバーを省略した形で呼び出す使い方の両方ができます。

```
pi = Math.atan2(1, 1)*4;
include Math
pi2 = atan2(1, 1)*4
```

## 定数
### 円周率(PI)
```
p Math::PI
# => 3.141592654
```

### 自然対数の底(E)
```
p Math::E
# => 2.718281828
```
